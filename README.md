# Paper: New Tab Notebook
## Current Functionality
**Bold** | `CTRL + B`
<br>
<br>
*Italicize* | `CTRL + I` 
<br>
<br>
<ins>Underline</ins> | `CTRL + U` 
- [x] Checkboxes | `[ ] + space`
* Bullet points | `- + space`
